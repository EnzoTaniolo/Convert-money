# Convert-money
Projeto de um simples conversor de moedas para práticas dos estudos em Javascript Web.<br>
Utilizei as tecnologias:
* HTML
* CSS
* JAVASCRIPT

O projeto é Responsivo, e converte de Real para Dólar, Euro e Bitcoin.
